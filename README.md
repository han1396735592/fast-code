# fast-code

一个代码生成框架

## 如何使用

请参考 `fast-code-admin`
1. 引入`stater`
    ```xml
    <dependency>
        <artifactId>fast-code-starter-web</artifactId>
        <groupId>cn.qqhxj</groupId>
        <version>0.0.1-SNAPSHOT</version>
    </dependency>
    ```

2. 配置 `fast-code`
    ```yaml
    fast-code:
      configPath: config
    
    ```

3. 启动访问

    打开浏览器 [fast-code](http://localhost:8080/fastCode)
